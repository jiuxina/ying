## 🎯 萤的独特优势

### 1. 🎨 极致个性化
- **唯一支持自定义字体上传** - 让你的倒数日独一无二
- **粒子特效系统** - 樱花、雨滴、萤火虫、雪花
- **12种主题色** - 覆盖各种审美偏好

### 2. 📸 完整的记忆系统
- **事件相册** - 为每个重要日子添加照片记录
- **时间线展示** - 回顾美好时光
- **回忆提醒** - "去年的今天"功能

### 3. 🔔 智能提醒系统
- **多阶段自动提醒** - 1天、3天、7天、30天、90天
- **智能提醒模式** - 根据重要性自动调整
- **提醒历史记录** - 查看所有提醒

### 4. 📊 数据可视化
- **进度环** - 直观的倒计时进度
- **年度网格** - 365天一目了然
- **统计分析** - 了解你的时间分布

### 5. 🔒 隐私保护
- **生物识别锁** - 指纹/Face ID
- **私密事件** - 不在小部件显示
- **数据加密** - 安全可靠

### 6. 🚀 性能优化
- **轻量级** - 内存占用 < 100MB
- **快速启动** - < 2秒
- **低崩溃率** - < 0.1%

### 7. 💰 完全免费开源
