# 萤 v2.0 - 新功能预览

## 🎉 即将推出

萤 v2.0 将带来一系列强大的新功能，让您的倒数日体验更加完善！

---

## ✨ 核心新功能

### 🔔 智能提醒系统

**告别忘记重要日子！**

- **多阶段提醒**: 自动在 1天、3天、7天、30天、90天前提醒您
- **智能模式**: 根据事件重要性自动调整提醒频率
- **自定义规则**: 完全按照您的习惯设置提醒
- **提醒历史**: 查看所有已发送的提醒记录

### 📸 事件相册

**留住每一个美好瞬间**

- 为每个事件添加照片（最多50张）
- 时间线展示，回顾美好时光
- 文字日记，记录心路历程
- "去年的今天"回忆提醒

### 🔒 隐私保护

**您的秘密，只有您知道**

- 指纹/Face ID 解锁应用
- 标记事件为私密（不在列表和小部件显示）
- PIN 码备用方案
- 可配置的自动锁定时间

### 📊 进度可视化

**让时间看得见**

- **圆形进度环**: 直观显示倒数进度
- **年度网格**: 365天一目了然（类似 GitHub 贡献图）
- **月度进度**: 清楚看到本月安排
- **统计卡片**: 事件分布和时间分析

### 📝 事件模板

**快速创建，一键搞定**

- 18个内置模板：生日、纪念日、考试、节日等
- 农历日期自动转换
- 年龄自动计算
- 保存常用配置为模板
