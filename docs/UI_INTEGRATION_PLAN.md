## 📍 集成点规划

### 1. 高级提醒系统 🔔

#### 集成位置
- **事件编辑页面** (`AddEditEventScreen`)
  - 在现有提醒设置下方添加"高级提醒"选项
  - 点击展开高级提醒配置面板

#### UI 设计
```
┌─────────────────────────────────┐
│ 提醒设置                         │
│ ├─ 基础提醒（现有功能）           │
│ │   └─ [选择提醒时间]            │
│ │                               │
│ └─ 高级提醒 ✨ NEW               │
│     ├─ [开启高级提醒] Switch     │
│     ├─ 智能提醒模式 [自动/手动]   │
│     ├─ 多阶段提醒                │
│     │   ├─ ☑ 提前 1 天          │
│     │   ├─ ☑ 提前 3 天          │
│     │   ├─ ☑ 提前 7 天          │
│     │   ├─ ☑ 提前 30 天         │
│     │   └─ ☑ 提前 90 天         │
│     └─ 自定义规则 [+ 添加]       │
└─────────────────────────────────┘
```

#### 导航流程
1. 用户创建/编辑事件
2. 展开"高级提醒"设置
3. 配置多阶段提醒或智能模式
4. 保存事件时自动创建 `AdvancedReminder` 记录

#### 代码修改
- `lib/screens/add_edit_event_screen.dart`
  - 添加高级提醒配置 UI
  - 集成 `AdvancedReminderService`
  - 保存时创建高级提醒记录

### 2. 事件相册/故事 📸

#### 集成位置
- **事件详情页面** (`EventDetailScreen`)
  - 在事件信息下方添加"回忆相册"标签页
  - 支持滑动切换（信息 / 相册）

#### UI 设计
```
┌─────────────────────────────────┐
│ [事件标题]                       │
│ [倒计时显示]                     │
│                                 │
│ [信息] [相册] [统计]  ← 标签页   │
│ ─────────────────────────────── │
│                                 │
│ 📷 照片时间线                    │
│ ├─ 2026-04-30                   │
│ │   └─ [照片1] [照片2]          │
│ ├─ 2026-04-15                   │
│ │   └─ [日记] "今天..."         │
│ └─ [+ 添加回忆]                 │
└─────────────────────────────────┘
```

#### 操作流程
1. 用户打开事件详情
2. 切换到"相册"标签
3. 点击"+"添加照片或日记
4. 查看时间线展示的所有回忆

#### 代码修改
- `lib/screens/event_detail_screen.dart`
  - 添加 TabBar（信息/相册）
  - 集成 `MemoryTimeline` 组件
  - 添加照片/日记上传功能

### 3. 隐私与安全 🔒

#### 集成位置
- **设置页面** (`SettingsScreen`)
  - 添加"隐私与安全"设置项
- **应用启动**
  - 检查生物识别锁状态

#### UI 设计
```
设置页面：
┌─────────────────────────────────┐
│ 设置                            │
│ ├─ 外观                         │
│ ├─ 通知                         │
│ ├─ 隐私与安全 🔒 NEW            │
│ │   ├─ 生物识别锁 [开启]        │
│ │   ├─ PIN 码设置               │
│ │   ├─ 自动锁定时间 [5分钟]     │
│ │   └─ 私密事件管理             │
│ └─ 云端同步                     │
└─────────────────────────────────┘

事件编辑：
┌─────────────────────────────────┐
│ 事件设置                        │
│ ├─ ...                         │
│ └─ ☑ 标记为私密事件 🔒          │
└─────────────────────────────────┘
```

#### 功能流程
1. 用户在设置中开启生物识别
2. 设置备用 PIN 码
3. 创建事件时可标记为私密
4. 应用启动/查看私密事件时需验证

#### 代码修改
- `lib/screens/settings/settings_screen.dart`
  - 添加"隐私与安全"入口
- `lib/main.dart`
  - 启动时检查生物识别锁
  - 显示验证对话框
- `lib/screens/add_edit_event_screen.dart`
  - 添加"私密事件"开关

### 5. 事件模板系统 📋

#### 集成位置
- **首页浮动按钮** 或 **首页顶部**
  - 添加"从模板创建"入口
- **事件编辑页面**
  - 保存事件时可选择"保存为模板"

#### UI 设计
```
首页浮动按钮：
        ┌─────┐
        │  +  │ ← 主按钮
        └─────┘
          ├─ 新建事件
          └─ 从模板创建 ✨ NEW

模板画廊：
┌─────────────────────────────────┐
│ 选择模板                        │
│                                 │
│ 🎂 生日                         │
│ ├─ 标题: 生日                   │
│ ├─ 分类: 生日                   │
│ └─ 提醒: 提前7天                │
│                                 │
│ 💑 纪念日                       │
│ ├─ 标题: 纪念日                 │
│ └─ 重复: 年度                   │
│                                 │
│ [+ 自定义模板]                  │
└─────────────────────────────────┘
```

#### 操作流程
1. 用户点击"+"按钮
2. 选择"从模板创建"
3. 浏览模板分类
4. 选择模板快速创建事件
5. 修改具体信息后保存

#### 代码修改
- `lib/screens/home_screen.dart`
  - 扩展浮动按钮为菜单
  - 或添加顶部快捷入口
- 导航到 `TemplateGalleryScreen`
- `lib/screens/add_edit_event_screen.dart`
  - 添加"保存为模板"选项
